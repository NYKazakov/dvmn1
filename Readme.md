# Погода
Простой скрипт для получения погоды прямо в строке терминала.

Весь конфиг захардкожен так что получить прогноз можно только на три ближайших дня для трех локаций: *Лондон*, *аэропорт Шереметьево* и *Череповец*.
Только день и ночь, черно-белое отображение, единицы измерения в системе СИ, скорость ветра в м/c.

## запуск
Для запуска требуется установленный python с модулем requests.
Запускается командой 
```
 python lesson1.py
```
